This interactive tool aims to supplement a semester-long project in which I forecasted the 2020 Presidential Election. The below interactive tools present and contextualize my predictions in a more visual manner than I did in my blog posts. Feel free to scroll through and enjoy! This page is largely inspired by the FiveThirtyEight forecast. Nate Silver, if youre seeing this, please hire me! See my full election blog and forecast write-up here.
